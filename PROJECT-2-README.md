# PROJECT-2: Iris Classification using KNN 🌸

**DecodeLabs Internship Project 2**

A complete Supervised Machine Learning pipeline using the famous Iris dataset and K-Nearest Neighbors (KNN) algorithm.

---

## 📋 Project Overview

This project demonstrates the full **Machine Learning Pipeline**:
- Data Loading
- Feature Scaling
- Train-Test Split
- Model Training (KNN)
- Prediction & Evaluation
- Visualization (Confusion Matrix)

**Key Concepts Covered:**
- Supervised Learning
- Feature Scaling with StandardScaler
- KNN Algorithm
- Model Evaluation Metrics (Accuracy, F1 Score, Confusion Matrix)

---

## 🚀 Features

- Complete end-to-end ML pipeline
- Detailed step-by-step console output
- Professional visualizations (Confusion Matrix)
- High accuracy on Iris dataset (~95-97%)
- Clean, well-documented Object-Oriented code
